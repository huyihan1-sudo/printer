# 🖨️ Retro Scribe — Retro Polaroid Typewriter

> *Analog Soul in Digital Form*

一款复古拍立得打字机网页应用，把你的文字瞬间"打印"成复古拍立得风格的卡片。

## ✨ 功能特性

- 📝 输入最多 140 字的文字内容
- 📸 点击"打印拍立得"触发打印动画
- 🖼️ 打印结果以复古拍立得相片风格展示
- 🌑 暗黑风格 UI，极简美学

## 🚀 在线体验

👉 **[点击这里立即体验](https://huyihan1-sudo.github.io/printer/)**

